# LIFE GAME 1차 개발 범위

1차 개발의 목표는 전체 게임 완성이 아니라 핵심 도메인 구조와 플레이 가능한 최소 흐름을 세우는 것입니다.

## 포함

- 사용자 초기 개인 자본금 50,000,000원
- 기업 설립, 지분 투자, 취업, 개인 자산 매입
- Event Log 원본 사건 기록
- News mock 생성
- 관리자 NPC 제어 타입과 기본 handler

## 제외

- 완전한 NPC AI 의사결정 엔진
- 상세 정산 공식
- 상장, 채권, 인수합병, 상세 세금
- 주주총회 상세 투표
- 가족/상속 상세 시뮬레이션
